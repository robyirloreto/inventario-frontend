# Sistema de Gestión de Inventario (Frontend) 💻

Interfaz de usuario moderna para el control de stock, desarrollada con **React** y **Vite**.

## Características
* **Visualización en Tiempo Real:** Cards dinámicas que muestran el stock actual.
* **Alertas Visuales:** Indicadores de bajo stock automáticos.
* **Diseño Responsivo:** Adaptable a diferentes tamaños de pantalla.
* **Integración API:** Conexión fluida con backend de Node.js mediante Axios.

## Tecnologías
* **React.js**
* **Vite**
* **Lucide React** (Iconografía)
* **Axios** (Peticiones HTTP)
